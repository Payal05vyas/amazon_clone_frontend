Amazon Clone (Frontend)

This project is a frontend clone of the Amazon homepage built using HTML and CSS. It replicates the core layout and design elements of the platform, including the navigation bar, hero section, and product category cards.

Features

* Responsive navigation bar with search UI
* Hero section with promotional banner
* Product category cards using Flexbox and CSS Grid
* Clean and structured layout

Tech Stack

* HTML
* CSS

Purpose

This project was created to practice and strengthen frontend development skills, focusing on layout design, styling, and component structuring.

Deployed using GitHub Pages
